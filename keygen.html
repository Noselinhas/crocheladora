📋 Como configurar o Firebase (5 min)

Siga os passos abaixo. Só precisa fazer uma vez:

  • 1
    Abra console.firebase.google.com e faça login com sua conta Google. Clique em "Criar projeto" → dê o nome crocheladora → clique em Continuar até criar.
  • 2
    No menu lateral esquerdo clique em "Compilar" → "Realtime Database". Em seguida, clique em "Criar banco de dados". Escolha a localização Estados Unidos e inicie em modo de teste.
  • 3
    Após criado, você verá uma URL como https://crocheladora-abc12-default-rtdb.firebaseio.com. Copie essa URL e cole no campo acima.
  • 4
    Ainda no Firebase, clique na aba "Regras" e substitua todo o conteúdo pelas regras abaixo. Isso impede que uma chave seja ativada duas vezes:
    { "rules": { "activations": { "$key": { ".read": true, ".write": "!data.exists()" } } } }
  • 5
    Clique em "Publicar" para salvar as regras. Pronto! Agora cole a URL no campo acima e clique em Salvar.
  • 6
    Último passo: Ainda precisa informar a URL para o app. Abra no GitHub o arquivo js/license.js, clique no lápis (editar) e encontre a linha com localStorage.getItem('croc_firebase_url'). O app já lê a URL que você salvou aqui — ela fica compartilhada automaticamente!
